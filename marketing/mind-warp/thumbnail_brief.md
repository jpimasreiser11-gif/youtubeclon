# Thumbnail Brief - Mind Warp

## Fixed visual principles
- High contrast, one dominant idea per frame.
- Max text: 2-4 words, mobile-first readability.
- Keep channel palette/preset consistency.

## Format-specific brief
### Format: Brain glitch
- Hero visual: Close-up face + neural overlay + visual distortion
- Thumbnail copy: Text: 'YOUR BRAIN LIES'
- Composition: hero element on left third, tension/reveal element on right.
- Delivery: 1280x720, verify readability at 10% zoom.

### Format: Experiment reveal
- Hero visual: Lab setup + participant reaction
- Thumbnail copy: Text: 'TESTED'
- Composition: hero element on left third, tension/reveal element on right.
- Delivery: 1280x720, verify readability at 10% zoom.

### Format: Daily trap
- Hero visual: Common situation with highlighted mistake
- Thumbnail copy: Text: 'YOU DO THIS'
- Composition: hero element on left third, tension/reveal element on right.
- Delivery: 1280x720, verify readability at 10% zoom.

## Production workflow
1. Build 3 variants per video (A/B/C).
2. Keep title constant; change one visual variable per variant.
3. Save final as `thumbnail_final.jpg` and alternates in `thumbnail_variants/`.
4. If early internal CTR is weak, iterate short copy + contrast in first 24-72h.
