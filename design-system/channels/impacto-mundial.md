# Channel Design System - Impacto Mundial

- Language: ES
- Core vibe: historical mystery + forbidden knowledge.
- Primary palette:
  - primary `#C8A829`
  - secondary `#0A0A0A`
  - accent `#8B0000`

## Tone and copy
- Tense, reveal-driven, dramatic pauses.
- Headlines should imply classified/hidden truths.

## Thumbnail direction
- Archetype: `dramatic_aerial`
- Keywords: maps, declassified docs, ruins, high-contrast red callouts.

## UI identity layer
- KPI highlight: gold.
- Alert/warning overlays: deep red.
- Recommended icon set: map pin, archive, lock.
