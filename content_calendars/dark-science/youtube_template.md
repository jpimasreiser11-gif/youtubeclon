# YouTube Script Template - Dark Science

## Inputs
- Scientific question:
- Current evidence:
- Competing interpretations:
- Visual analogy set:

## Structure (9-15 min)
1. **HOOK**
   - "Measured fact" + "why this is unsettling".
2. **PROMISE**
   - What viewers will understand by the end.
3. **EXPLANATION ARC**
   - Observation
   - Mechanism hypothesis
   - Limits and uncertainty
   - Real-world implication
4. **CTA INTEGRATED**
   - Invite to the next phenomenon in the same arc.
5. **CLOSING**
   - Wonder + caution + open scientific question.

## Retention
- Add one scale analogy each major block.
- Use cliffhanger transitions every 35-50s.
- Alternate macro (cosmic) and micro (human-scale) framing.

## Anti-ban / Accuracy
- Do not state unconfirmed hypotheses as fact.
- No fabricated citations.
- Explicitly label uncertainty ranges.
- Keep educational framing, avoid fearmongering.
