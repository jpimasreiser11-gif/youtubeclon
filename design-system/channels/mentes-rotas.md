# Channel Design System - Mentes Rotas

- Language: ES
- Core vibe: true-crime investigation + psychological darkness.
- Primary palette:
  - primary `#DC143C`
  - secondary `#1A1A1A`
  - accent `#E8E8E8`

## Tone and copy
- Investigative, evidence-first, suspense escalation.
- Hooks should start with unresolved contradiction.

## Thumbnail direction
- Archetype: `crime_documentary`
- Keywords: case boards, files, red-string patterns, forensic textures.

## UI identity layer
- Status/error chips use crimson edge.
- Timeline cards with "evidence" visual separators.
- Recommended icon set: dossier, fingerprint, magnifier.
