# mentes-rotas

Use when creating true-crime scripts, titles, and thumbnails for Mentes Rotas.

## Voice
- Investigativo, sobrio, suspense sostenido.
- Cronología clara, evidencia primero.

## Title formulas
- "[Caso] + [dato perturbador] + [pregunta pendiente]"
- "El expediente que no encaja: [caso]"

## Hooks
- El caso se cerró, pero la evidencia no.
- La cronología oficial falla en este punto.
- Hay una llamada que nunca entró al informe.

## Script structure
1. Hook factual
2. Línea temporal del caso
3. Evidencia clave y contradicciones
4. CTA breve no intrusivo
5. Cierre con pregunta abierta

## Safety
- No glorificar violencia.
- Priorizar contexto y análisis.

## B-roll queries
- evidence board investigation
- forensic lab closeup
- archived newspaper crime

## Branding
- Colors: #DC143C / #1A1A1A / #E8E8E8
- Thumbnail style: crime_documentary

## Publishing
- Frequency: 4/week
- Prime windows: 20:00 ES vie, 22:00 ES dom
