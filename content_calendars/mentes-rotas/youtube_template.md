# YouTube Script Template - Mentes Rotas

## INPUTS
- Caso:
- Línea temporal (5 hitos):
- Evidencia confirmada:
- Zona gris / incertidumbre:

## STRUCTURE (10-15 min)
### 1) HOOK
- Abrir con el punto más inquietante del expediente.
- Sin saludos.

### 2) PROMESA
- Qué piezas vamos a conectar.
- Qué mito del caso vamos a corregir.

### 3) RECONSTRUCCIÓN CRONOLÓGICA
Bloques sugeridos:
- Antes del crimen
- Momento crítico
- Investigación inicial
- Giro forense o testimonial
- Estado actual del caso

Cada bloque incluye:
- Hecho confirmado
- Interpretación prudente
- Pregunta de retención para el siguiente bloque

### 4) CTA INTEGRADA
- "Si quieres la línea temporal completa con documentos, suscríbete".

### 5) CIERRE
- Resumen de hechos vs hipótesis.
- Cierre respetuoso orientado a aprendizaje investigativo.

## RETENTION
- Abrir loops cada 45-60s.
- Cambios de ritmo con mini-resúmenes.
- Repetir nombres/fechas clave para claridad.

## ANTI-BAN
- No glorificar violencia ni describir detalles gráficos.
- No acusar sin respaldo documental público.
- Enfatizar "presunto" cuando aplique.
- Tono analítico, no sensacionalista.
