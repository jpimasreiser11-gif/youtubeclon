# youtube-seo-master

Use for titles, descriptions, tags, thumbnails, and CTR optimization.

## Title principles
- Keyword in first 1-3 words where possible.
- 45-60 chars ideal for CTR.
- One strong curiosity gap, no generic clickbait templates.

## Description structure
1. 2 lines with core promise + keyword
2. concise value bullets
3. CTA
4. hashtags + semantic keyword variants

## Tags strategy
- Blend:
  - core intent terms
  - semantically related terms
  - trend-tail terms
- Keep relevance strict by channel niche.

## Thumbnail CTR rules
- One focal point.
- High luminance contrast.
- 2-5 word overlay max.
- Consistent channel color grammar.

## Workflow
1. Draft metadata
2. Humanize and remove AI phrasing
3. Check niche language fit
4. Publish and track CTR/retention deltas
