# Cross Promotion - Wealth Files

## Objective
Use all 6 channels as a discovery network while keeping editorial fit.

## Global rules
1. Max one strong cross-promo per video.
2. Only promote a channel when overlap is clear.
3. Use tracked links (`?ref={SOURCE_CHANNEL}`).
4. Rotate placements: verbal mention, pinned comment, end screen, community post.
