```md

- {BULLET_2_APRENDIZAJE}
- {BULLET_3_APRENDIZAJE}

{CHAPTERS_BLOCK}

Fuentes / recursos:
{SOURCES_BLOCK}

---
🔗 Herramientas y recursos recomendados
{AFFILIATE_BLOCK_1}
{AFFILIATE_BLOCK_2}
{AFFILIATE_BLOCK_3}

---

{NEXT_VIDEO_URL}

# {HASHTAG_1} #{HASHTAG_2} #{HASHTAG_3}
```

## Bloques de afiliados (placeholders listos)
- `{AFFILIATE_BLOCK_1}`: Producto principal del episodio.
- `{AFFILIATE_BLOCK_2}`: Alternativa económica o free trial.
- `{AFFILIATE_BLOCK_3}`: Recurso complementario (plantilla, libro, curso).

Formato sugerido por bloque:
```md
- Recurso: {NOMBRE_RECURSO}
- Para qué sirve: {BENEFICIO_DIRECTO}
- Enlace: {URL_AFILIADO_O_DIRECTO}
```

## Reglas operativas
1. Primera línea con gancho y tema, sin saludo largo.
2. Máximo 3 bloques de afiliado para mantener confianza.
3. Priorizar recursos realmente usados en el contenido.
4. Revisar que disclosure esté siempre presente.
5. Validar links antes de publicar.

## Ideas de recursos frecuentes para este canal
- libros de criminología, cursos de investigación, audiolibros de casos reales
