# Brand Assets (Phase 3)

Contenido generado para fase visual:

- `logos/*.svg`: logos base por canal.
- `prompts/banana-logo-prompts.txt`: prompts listos para Banana.
- `prompts/banana-thumbnail-prompts.txt`: batches de thumbnails (18 total objetivo).
- `prompts/banana-dashboard-landing-prompts.txt`: assets de producto/landing.
- `presets/channel-presets.json`: presets de estilo por canal para regeneración.

Nota: los SVG son fallback locales para mantener el sistema operable sin bloqueo de generación externa.
