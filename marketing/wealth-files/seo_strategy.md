# SEO Strategy - Wealth Files

## Channel SEO objective
- Own search intent around wealth systems and case-study finance.
- Match title/description/tags to how the English-speaking aspirational finance viewers searches on YouTube.
- Favor informational + narrative intent, avoid empty hype wording.

## Main YouTube keywords (core)
- wealth building
- money systems
- investing strategy
- financial case study
- cash flow strategy
- wealth psychology
- business breakdown
- asset allocation
- financial freedom plan
- high income skills

## Long-tail keywords (specific intent)
- wealth building systems most people ignore
- financial case studies with actionable lessons
- how cash flow systems create long-term wealth
- asset allocation strategy for beginners
- money habits that actually compound
- wealth psychology behind smart decisions
- how businesses use leverage to grow
- financial freedom roadmap with realistic steps
- why income alone does not build wealth
- hidden mechanisms behind net worth growth

## Weekly execution (actionable)
1. Pick 1 core keyword + 2 long-tail keywords per video.
2. Put the core keyword near the start of the title and first description line.
3. Repeat semantic variations in intro, summary, and CTA blocks.
4. Add 5-8 tags: 2 core, 3 long-tail, 1-3 timely/context tags.
5. Review CTR/retention at 72h and update title/thumbnail when needed.

## Per-video checklist
- [ ] Primary keyword appears naturally in title.
- [ ] First description line contains promise + keyword.
- [ ] 2-3 long-tail terms used without stuffing.
- [ ] Closing hashtags match topic and audience intent.
