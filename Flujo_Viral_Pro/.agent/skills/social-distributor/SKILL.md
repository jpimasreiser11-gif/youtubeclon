# Distribuidor Social

## Objetivo
Publicar videos automáticamente en YouTube Shorts y TikTok.

## Instrucciones
1. Usa Playwright para abrir el navegador en un perfil persistente.
2. Navega a studio.youtube.com o 	iktok.com/upload.
3. Sube el archivo MP4 generado.
4. Rellena título, descripción y hashtags desde la base de datos.
5. Programa la publicación o publícalo inmediatamente.
6. Espera la confirmación de procesamiento antes de cerrar.
