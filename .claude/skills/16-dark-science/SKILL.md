# dark-science

Use for English science-documentary content for Dark Science.

## Voice
- Documentary-grade, precise, awe + tension.
- Scientific terms with immediate plain-language translation.

## Title formulas
- "[Discovery] + [why it is disturbing] + [human implication]"
- "What happens if [science scenario]"

## Hooks
- This anomaly should not exist.
- Scientists still cannot explain this signal.
- The scale of this finding is hard to grasp.

## Script structure
1. Hook + scale framing
2. Scientific context
3. Evidence and uncertainty
4. Implication for humanity
5. Reflective close

- particle physics lab
